# **App Name**: Canaan Connect

## Core Features:

- Homepage Structured Content Display: Showcase a large hero banner, 'About' overview, 'Vision & Mission', 'Core Values', 'Facilities', 'Leadership' profiles, 'Achievements' highlights, and an 'Admissions Call to Action' in clear, distinct sections.
- Essential Static Pages: Provide dedicated 'About', 'Admissions', and 'Contact' pages to house detailed information for parents and prospective students.
- Responsive Sticky Navigation: Implement a full-width sticky navigation bar with a logo placeholder and primary menu items ('Home', 'About', 'Academics', 'Facilities', 'Contact') for intuitive access across all devices.
- Inquiry Management System: Enable users to submit inquiries (name, phone, message) via a 'Contact' form, with all submissions securely stored in a Firestore 'enquiries' collection, including a timestamp.
- Dynamic Admissions Information: Prominently display key admissions details like 'Admissions Open 2026–27', 'No Donation', along with actionable buttons like 'Apply Now' and 'Book a Visit' to facilitate enrollment.
- Visual School Gallery: Present a responsive image grid in the 'Gallery' section, offering a visual overview of school life, events, and facilities.
- AI Admissions Guidance Tool: Integrate an AI tool on the admissions page to intelligently answer common prospective parent queries about enrollment, school policies, and processes based on available website content.

## Style Guidelines:

- Primary color: A deep, calming blue (#2259C3) that conveys professionalism, trust, and academic excellence, making it suitable for headlines and important interactive elements.
- Background color: A very light, subtle bluish-gray (#F0F1F4) provides a clean, open canvas, maintaining a sophisticated yet approachable aesthetic for the main content.
- Accent color: A muted violet-blue (#8170DB) that harmonizes with the primary blue while offering enough contrast to highlight calls to action and secondary interactive elements without overpowering the design.
- The primary font for all text, including headlines and body, will be 'Inter' (sans-serif). Its modern, neutral, and highly legible design ensures clear communication and a clean, professional appearance throughout the site.
- Employ a consistent set of minimalist, line-art style icons. These will enhance clarity and modern appeal for features, core values, and facilities sections without adding visual clutter.
- Adopt a section-based content hierarchy with ample whitespace and padding. Implement a flexible grid system for card-based features (values, facilities) and a two-column layout for narrative content (vision, mission) to ensure optimal readability and modern aesthetics across all screen sizes. The header will be sticky, maintaining navigation accessibility.
- Incorporate subtle 'fade-in' animations for content sections as they scroll into view, and gentle hover transitions for interactive elements (buttons, cards). These minor details will contribute to a smooth, premium user experience without being distracting.
